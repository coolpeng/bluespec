# BlueSpec

A specification-driven BLE exploration tool for AI-native iOS development.

BlueSpec is an experimental iOS project exploring how OpenSpec and Superpowers can guide AI-assisted development of a real CoreBluetooth application.

## Goals

- build a minimal LightBlue-like BLE exploration app
- validate specification-driven AI development workflows
- explore state-machine-driven UI architecture

## Initial scope

- peripheral scanning
- device connection
- service discovery
- characteristic inspection
